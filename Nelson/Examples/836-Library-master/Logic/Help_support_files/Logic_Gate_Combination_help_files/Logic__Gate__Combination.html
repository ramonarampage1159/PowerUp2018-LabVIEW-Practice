Logic_Gate_Combination.vi

Help by FRC Team 836 - The RoboBees

This VI is a good trainer on how AND, OR, and XOR Boolean functions operate.

The user selects which Boolean to use.

"True" values are loaded into the input arrays by clicking the buttons for Array A & B.

The output array will display the Boolean operation result.

The "Different Input Array Length" flag instructs how the VI handles situatioons where the input arrays are not the same size.

Note: the arrays do not reset while in operation OR after restarting. The arrays must be reset using the "Reinitialize to default" operation

in the "Edit" menu.
