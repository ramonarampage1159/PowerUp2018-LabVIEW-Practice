Bump_Counter.vi

Description:

This subVI provides discrete stepped control of an output value relative to an initial reference value. The output is quantized at a user specified level.

Inputs:

REF - Reference value, and initial value of OUT prior to adding bump values.

RESET - Erases the bumps on the rising edge and returns OUT to the value REF, suspending further bumping while it is True.

The bumping function resumes when RESET becomes False.

CNTRL - Control input value which is monitored for transition point crossings. Crossing from the dead-band to over the +0.5 point

will increase OUT by one QUANTA. Transitioning over the -0.5 point decreases OUT by one QUANTA.

QUANTA - The discrete step value used to change OUT each time CNTRL crosses a transition point. Provides a designed level of precision for commanded changes to OUT.

Outputs:

OUT - Reference value modified by commanded count of quantum bumps.
