UpDown_Counter.vi

Help by FRC Team 836 - The RoboBees

Description:

This VI will maintain an integer COUNT, incremented by rising edges of UP and decremented by rising edges of DOWN.

Simultaneous UP and DOWN commands result in no count change, since this is an ambiguous command.

Inputs:

INITIAL - Initial value of COUNT and also value of COUNT upon RESET command

RESET - Boolean command, COUNT is reset to INITIAL on rising edge, and suspends counting while True.

Normal operation resumes on falling edge

UP - Boolean up count signal, operates on rising edge

DOWN - Boolean down count signal, operates on rising edge

Outputs:

COUNT - Outputs and holds current count value
